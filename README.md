## Interval-Machine-test

## Installation

# Clone the repository
git clone https://github.com/yourusername/your-repo.git

# Navigate to the project directory
cd Interval-Machine-test

# Install dependencies for both server and client
# for server
cd server 
npm i
# for client
cd client
npm i

# Start the client
cd client
npm run dev

# Start the server
cd server
npm start
